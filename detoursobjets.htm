Installation multimédia « Détours d'objets » au Centre Pompidou

2007 pour Numeriscausa



Type de projet : Stage

Technologie : C++, librairies SDL, OPENGL et SGE, WINDOWS

Travail : Développement, réseaux, installation, configuration

Description :

Diaporama diffusant en permanence et de manière aléatoire les derniers empilements conçus par les visiteurs. Les images sont communiquées par réseaux de l'installation vers le diaporama.

Tâches :

  • Développeur C++ d'un diaporama diffusant en permanence et de manière aléatoire les dernières images conçus par les visiteurs d'une installation multimédia se trouvant au centre Pompidou.

  • Installation et configuration de la synchronisation FTP.
