Reconnaissance de caractères et de symboles sur verre gravé

Stagiaire – Maîtrise Dijon

3Cconcept : R&D Électronique et Informatique

 



 

Type de projet : Stage

Technologie : DELPHI, WINDOWS, caméra

Travail : Recherche et développement, traitement d'image

Description :

C'est'un système de reconnaissance de symboles gravés sur les verre par caméra dans le but de tracer les verres.

Tâches :

Conception d'un logiciel en DELPHI de reconnaissance de symboles gravés sur des verres par caméra.
Programmation d'une bibliothèque de traitement d'image dans le but de reconnaître les symboles.
