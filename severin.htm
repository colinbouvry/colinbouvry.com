Installation interactive "Changing Herakles" de l'artiste Severin Krön

2011 pour Severin Krön



Type de projet : Freelance

Technologie : Processing, JAVA

Travail : Développement, installation

Description :

Logiciel de l'artiste Severin Krön d'art permutable.
Les pièces sont composées de carrés pouvant être  tournés ou déplacés à volonté.  Quelque soit le positionnement des modules les liens de la composition  persistent.
Avec 9 carrés formez  plus de 95 milliards de compositions !

http://www.severin-kron.com/

Non seulement les images sont tournables et permutables mais le son est aussi permutable.
Avec ce logiciel vous pouvez aussi sauver la configuration actuelle pour la reprendre lors d'un nouveau rédémarrage du logiciel, vous pouvez exporter une image créée pour l'utiliser, vous pouvez zoomer et naviguer dans la composition, vous pouvez changer les couches des images et vous pouvez permuter et tourner aléatoirement les images de façon automatique.

Tâches :

  • Développeur en JAVA avec processing avec les librairies suivantes : Minim pour l'audio, Opengl pour le graphisme, ProXML pour l'XML et ControlP5 pour l'IHM.

  • Installation lors d'une exposition de Severin Krön.
