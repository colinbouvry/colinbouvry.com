Nébulosa au salon international du meuble à Milan

2007 pour Numeriscausa



Type de projet : Stage

Technologie : C++, libraries OpenCV, SDL et Open Sound Control, WINDOWS

Travail : Développement, traitement d'image, installation, configuration

Description :

Une borne avec une surface sensible interagissant via une communication wifi avec une nébuleuse sous forme de particules.

Tâches :

  • Développeur C++ d'une borne interactive qui interagit par effleurement de surface (Traitement d'image en temps réel) pour la salon international du meuble de Milan.

  • Développeur C++ des communications UDP entre les différents ordinateurs.

  • Installateur (configuration, réglage).
